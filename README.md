# osu-mania-acc-calculator-for-SV1-&-SV2
A calculator for calculating Accs in scoreV1 &amp; scoreV2.  
## How to use this calculator  
Step 1: Input the hitcount of each judgement. (example: 4000 2000 200 100 50 60)  
Step 2: Get Accs of ScoreV1 and ScoreV2
